===============================================================
To run the code:

	make run
===============================================================
The calculator transformations:

Key '=' 	  : Zoom in 
Key '-' 	  : Zoom out
Key 'A' 	  : Rotate anti-clockwise
Key 'Z' 	  : Rotate clockwise
Key 'Left arrow'  : Translate left
Key 'Right arrow' : Translate right
Key 'Up arrow'    : Translate up
Key 'Down arrow'  : Translate down

===============================================================
Optional

Key 'Q' : To rotate every single button about their own centers
Key 'W' : To rotate the entire calculator about the origin
===============================================================



Errors:

The first button is clicked as soon as the program starts running.
